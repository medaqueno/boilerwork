## Permissions

These are the specific permissions for this context to added that can be found in ==\App\Core\Shared\Domain\ValueObjects\PermissionsProvider==

```
CAN_MANAGE_TENANTS;
CAN_EDIT_TENANT;
CAN_MANAGE_BRANCHES;
CAN_EDIT_BRANCH;
CAN_MANAGE_USERS;
CAN_EDIT_USER;
````
